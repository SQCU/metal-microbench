## Decisions visible in this artifact

- Reused existing 500 captures; no new rollouts (avoiding dense-grid B2)
- Added 1 Gemma-judged feature (trope_density+predictability_given_opener, 2 axes × 2 cards = 4 axes) + 13 free surface features × 2 cards = 26 axes = 30 total per bio
- Per (bio, card) values averaged across 10 openers from canonical alternate_greetings
- Z-scored before PCA so unit-scale (chars) vs ratio (frac_caps) features are comparable
- Pairwise distance metric: euclidean in z-scored feature space (= euclidean in full PC space)
- Cluster-separation threshold 0.7 / 1.0 are heuristic; intent is to highlight feature-design candidates (collapsed clusters), not to make hard rejections
